# House Price Prediction using XGBoost

## Overview
This project predicts house prices using the XGBoost regression algorithm based on various input features such as area, location, and other property attributes.

##  Steps Performed
- Data preprocessing and cleaning using Pandas
- Exploratory Data Analysis (EDA)
- Feature selection and handling missing values
- Applied XGBoost Regressor for prediction
- Evaluated model performance using regression metrics

## Tech Stack
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- scikit-learn
- XGBoost

## Output
- Predicted house prices based on input features
- Model evaluation using metrics like RMSE
